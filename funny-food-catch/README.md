# 🎈 Funny Food Catch - Kids Game

A simple, funny, and colorful game for kids! Catch falling foods with your basket while avoiding the stinky ones.

## 🎮 How to Play

1. **Move your basket** with mouse or touch
2. **Catch yummy foods** 🍕🍔🍦 - get points!
3. **Avoid bad foods** 💩🤢 - lose points and lives!
4. **Catch stars** ⭐🌈 - bonus points!
5. **Don't miss good foods** - you lose a life!

## 🎯 Features

- **Simple controls** - Just move left and right
- **Funny emojis** - Kids love the silly faces
- **Combo system** - Catch multiple in a row for bonus points
- **Particle effects** - Fun explosions when you catch things
- **High score** - Saves best score locally
- **Mobile friendly** - Works on phones and tablets
- **Colorful design** - Bright and engaging for kids

## 🚀 Play Now

Open `index.html` in any web browser!

## 🎨 Customization

Kids can customize by editing the emoji arrays in the code:
```javascript
const goodFoods = ['🍕', '🍔', '🍦', ...]; // Add more yummy food!
const badFoods = ['💩', '🤢', ...];        // Add more silly things!
```
