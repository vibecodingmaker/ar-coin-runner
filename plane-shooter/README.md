# ✈️ Sky Defender - 2D Plane Shooting Game

A complete arcade-style plane shooter with multiple levels, weapon upgrades, and boss battles!

## 🎮 Features

### Core Gameplay
- **Full movement** - Fly your plane with keyboard arrows
- **Shooting** - Auto-fire or manual shooting
- **Multiple weapon levels** - Upgrade your weapons as you progress
- **Combo system** - Consecutive kills give bonus points

### Systems
- ❤️ **Life System** - 3 lives, lose one per hit
- 💰 **Coin System** - Collect coins for bonus points
- ⭐ **Score System** - Points for enemies, combos, and boss kills
- 🔫 **Weapon Upgrades** - Level up as you play

### Enemies & Bosses
- **3 Enemy Types** - Different sizes, health, and rewards
- **Consecutive Kill Bonuses** - 5x, 10x, 20x kill streak rewards
- **Boss Battles** - Epic boss at end of each level
- **Special Enemy Rewards** - Different enemy types give extra points

### Level System
- **Level 1** - Training Mission (20 enemies)
- **Level 2** - Danger Zone (30 enemies, faster enemies)
- **Future Levels** - More enemies, bosses, and challenges

## 🚀 Installation & Play

### Browser
1. Download `index.html` and `game.js`
2. Open `index.html` in any modern browser
3. Press "START MISSION"

### Commands
```bash
npm install -g serve
serve .
```

## 🎯 Controls

- **Arrow Keys** - Move plane
- **A/D** - Move left/right
- **W/S** - Move up/down
- **Space** - Shoot (auto-fire enabled)

## 📊 Scoring

- **Basic Enemy:** 10-30 points
- **Special Enemy:** +50-100 bonus
- **Coin:** 5 points each
- **Combo Bonus:** +20 for every 5 consecutive kills
- **Boss Kill:** +1000 base + 500 bonus
- **Coin Collection:** +5 points per coin
